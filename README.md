# Homebrew Tap

Homebrew formulae and casks for [@vasilievyakov](https://github.com/vasilievyakov) projects.

## Install

```bash
brew tap vasilievyakov/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `claude-sessions` | Native macOS session browser for Claude Code |

### Claude Sessions

```bash
brew install --cask vasilievyakov/tap/claude-sessions
```

Requires macOS 14 (Sonoma) or later.
